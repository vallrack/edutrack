
# EduTrack - Ciudad Don Bosco

Sistema profesional de control de asistencia y gestión de jornadas para la institución Ciudad Don Bosco.

## Características del Sistema

- **Identificación Digital**: Generación automática de carnets con código QR institucional.
- **Gestión de Jornadas**: Configuración de horarios por días de la semana y horas de cumplimiento.
- **Marcaje Inteligente**: Estación de escaneo pública para entradas y salidas rápidas.
- **Panel Administrativo**: Control total de docentes, sedes y especialidades.
- **Reportes Avanzados**: Exportación de datos a PDF y Excel para procesos de nómina.
- **Análisis con IA**: Resúmenes automáticos de patrones de asistencia mediante Genkit.

## Guía de Despliegue (Git)

Para subir este proyecto a tu repositorio de GitHub, abre tu terminal en la carpeta del proyecto y ejecuta los siguientes comandos:

```bash
# 1. Inicializar el repositorio local
git init

# 2. Añadir todos los archivos al seguimiento
git add .

# 3. Crear el primer commit
git commit -m "first commit - Sistema EduTrack Don Bosco"

# 4. Configurar la rama principal como 'main'
git branch -M main

# 5. Vincular con tu repositorio remoto en GitHub
git remote add origin https://github.com/vallrack/edutrack.git

# 6. Subir el código a GitHub
git push -u origin main
```
